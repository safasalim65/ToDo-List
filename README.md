# ToDo-List

A simple ToDo List application built using Django REST Framework.
This project provides APIs for creating, reading, updating, and deleting tasks (CRUD operations).

# Features

Create, read, update, and delete tasks.

Django REST Framework-based API.

Admin panel for managing tasks.

JSON responses for API endpoints.
